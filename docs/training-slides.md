# 시스템 개요

- **목적**: 해군 사상자(사망자/상이자) 관리 및 전공사상심사 기록 전산화
- **주요 기능**:
  - 인원 등록/조회/수정/삭제 (CRUD)
  - 공식 문서 7종 PDF 출력
  - 통계/현황 차트 및 보고서
  - 전체 활동 감사 로그 자동 기록
- **접속**: `https://<서버IP>:8443` (군 내부망 전용)

# 로그인 및 보안

- **초기 비밀번호**: 관리자 제공, 첫 로그인 시 변경 필수
- **계정 잠금**: 비밀번호 5회 연속 오류 시 자동 잠금
- **자동 로그아웃**: 30분 무조작 시 세션 만료
- **개인정보 보호**: 주민번호 마스킹 표시 (역할별 노출 단계 차등)

> 잠금된 계정은 관리자(ADMIN)만 해제 가능

# 메뉴 구성

- **사망자 현황** -- 사망자 CRUD + Excel

# 사망자 관리 (1/2)

## 목록 조회

- 좌측 메뉴 **"사망자 현황"** 클릭
- 검색 조건: 군구분, 군번, 성명, 생년월일, 계급, 소속
- [조회] 버튼으로 검색 실행
- 페이지당 10/20/50건 선택 가능

---

# 사망자 관리 (2/2)

## 등록

1. [등록] 버튼 > Modal 팝업
2. 필수: 군번, 성명, 주민번호, 계급, 소속, 사망구분
3. 주민번호 형식 자동 검증 + 중복 군번 자동 차단
4. [저장] 클릭

## 수정/삭제

- **수정**: 행 클릭 > Modal에서 수정 > [저장]
- **삭제**: MANAGER 이상만 가능, 삭제 사유 필수 입력

# 전공사상심사 관리 (1/2)

## 심사 등록

- 심사차수, 심사일자, 성명, 군번, 주민번호, 입대일자
- 소속, 병명, 소속부대 심사결과, 전공상 분류
- 동일인 복수 심사 가능 (차수별 관리)

## 심사 결과 자동 반영

- 심사 결과 저장 시 연계된 사망자/상이자 분류 자동 업데이트

---

# 전공사상심사 관리 (2/2)

## 심사 이력 조회

- 대상자별 심사차수 이력을 **타임라인**으로 시각화
- Drawer 패널에서 상세 내용 확인

## 보훈청 통보

- [보훈청 통보] 버튼 클릭 > 확인 Modal
- 4단계 상태 전이 완료 후에만 통보 가능
- 통보일자 자동 기록

# 문서 출력 (1/2)

## 출력 가능 문서 7종

| 문서명 | 대상 |
|--------|------|
| 국가유공자 확인서 (사망자용) | 사망자 |
| 국가유공자 확인서 (상이자용) | 상이자 |
| 전공사상심사결과서 | 심사 대상자 |
| 순직/사망확인서 | 사망자 |
| 사망자 현황 보고서 | 사망자 전체 |
| 상이자 현황 보고서 | 상이자 전체 |
| 전사망자 확인증 발급대장 | 사망자 전체 |

---

# 문서 출력 (2/2)

## 출력 절차

1. 대상자 선택 후 **문서 출력** 버튼 클릭
2. 문서 유형 선택
3. **발급 목적 입력** (필수 -- 감사 추적)
4. PDF 미리보기 확인
5. [인쇄] 또는 [다운로드]

> 모든 발급 이력은 자동 기록됩니다
> (발급자, 일시, 목적, 대상자)

# 관리자 기능 (ADMIN 전용)

## 사용자 관리

- 신규 계정 생성, 역할 부여/변경

## 감사 로그 조회

## 데이터 임포트

- Excel 일괄 등록 (사망자/상이자/심사)
- 자동 검증 + 오류 안내

## 감사 보고서

- 월별 PDF 감사 보고서 생성

# 업무 시나리오 1: 사망자 등록 후 확인서 발급

1. **사망자 현황** > [등록] > 정보 입력 > [저장]
2. 등록된 행에서 **문서 출력** 클릭
3. "국가유공자 확인서(사망자용)" 선택
4. 발급 목적 입력 > PDF 미리보기 > [인쇄]

> 전체 과정 소요 시간: 약 3~5분

---

# 업무 시나리오 2: 전공사상심사 결과 처리

1. **전공사상심사** > [등록] > 심사 정보 입력
2. [저장] > 사망자/상이자 분류 자동 반영 확인
3. 보훈청 통보 필요 시 > [보훈청 통보] 클릭
4. 확인 Modal > [확인] > 통보일 자동 기록

---

# 업무 시나리오 3: 기존 데이터 일괄 등록

1. **관리** > **데이터 임포트** 탭
2. 유형 선택 (사망자/상이자/심사)
3. Excel 파일 업로드
4. 검증 결과 확인
5. 오류 시: 오류 행 확인 > 수정 > 재업로드

> 대량 데이터 초기 적재 시 활용

# 자주 묻는 질문 (FAQ)

**Q: 비밀번호를 잊어버렸습니다**
A: 관리자(ADMIN)에게 비밀번호 초기화 요청

**Q: "권한이 없습니다" 메시지가 나옵니다**
A: 현재 역할에 해당 기능 권한 없음. 관리자에게 역할 변경 요청

**Q: 삭제한 데이터를 복구할 수 있나요?**
A: 논리 삭제이므로 데이터는 보존됨. 관리자에게 복구 요청

**Q: 30분마다 로그아웃됩니다**
A: 보안 정책상 30분 무조작 시 자동 로그아웃. 작업 중에는 유지됨
