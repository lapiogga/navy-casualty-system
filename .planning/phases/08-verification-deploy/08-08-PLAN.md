    버전: 1.0.0 | 작성일: 2026-04-04

    ## 1. 사전 요구사항
    - 운영체제: Linux (CentOS 7+, Ubuntu 20.04+)
    - Docker Engine 24.0+ 및 Docker Compose v2
    - 디스크: 최소 20GB 여유 공간
    - 메모리: 최소 4GB RAM
    - 포트: 8080(HTTP), 8443(HTTPS), 5432(DB, 내부만)

    ## 2. 패키지 전달 및 압축 해제
    - USB/보안매체로 navy-casualty-deploy-v1.0.0.tar.gz 전달
    - tar xzf 명령으로 압축 해제

    ## 3. Docker 이미지 로드
    - deploy.sh 자동 실행 또는 수동: docker load < images.tar.gz

    ## 4. 환경변수 설정
    - .env.example -> .env 복사
    - 각 변수 설명 (DB_PASSWORD, PII_ENCRYPTION_KEY 등)

    - 자체 서명: ./deploy/generate-cert.sh
    - CA 발급 인증서: certs/keystore.p12 교체 방법
    - .env에 SERVER_SSL_ENABLED=true 설정

    ## 6. 서비스 시작
    - ./deploy/deploy.sh 실행
    - 또는 수동: docker compose -f docker-compose.yml -f docker-compose.prod.yml up -d

    ## 7. 헬스체크 확인
    - curl http://localhost:8080/actuator/health
    - docker compose ps

    ## 8. 초기 설정
    - 기본 관리자 계정: admin / admin (첫 로그인 시 비밀번호 변경 강제)
    - /api/admin/data-check으로 코드 테이블 정합성 확인

    ## 9. 네트워크 구성도
    [사용자] --HTTPS:8443--> [Docker: App Container] --5432--> [Docker: DB Container]
    - 방화벽: 8443 포트만 외부 허용, 5432는 Docker 내부 네트워크만

    ## 10. 방화벽 규칙
    - INBOUND: 8443/tcp (HTTPS), 8080/tcp (HTTP, 리다이렉트용)
    - OUTBOUND: 없음 (에어갭)
    - DB 포트(5432) 외부 차단 확인

    - 디스크 사용량: df -h, docker system df

    ## 14. 업데이트 절차
    - 새 패키지 전달 -> docker compose down -> 이미지 교체 -> docker compose up

    |------|------|------|
    | healthcheck 실패 | 앱 비정상 | docker compose restart app |
    | Connection refused | 앱 미기동 | docker compose logs app 확인 |
    | OOM (OutOfMemoryError) | 메모리 부족 | JVM 힙 조정 (JAVA_OPTS) |
    | disk full | 디스크 부족 | 로그/백업 정리, docker system prune |
    | DB 연결 실패 | DB 컨테이너 다운 | docker compose restart db |

    per D-29: docs/build-docs.sh 생성. Markdown 문서 3종을 pandoc으로 PDF 변환하는 스크립트.

    ```bash
    #!/bin/bash
    # docs/build-docs.sh — Markdown -> PDF 변환 (pandoc 필요)
    set -euo pipefail
    SCRIPT_DIR="$(cd "$(dirname "$0")" && pwd)"
    OUT_DIR="$SCRIPT_DIR/pdf"
    mkdir -p "$OUT_DIR"

    # CJK(한글) 폰트 설정
    CJK_FONT="${CJK_FONT:-NanumGothic}"

    for md in deployment-guide.md operations-manual.md; do
      echo "Converting $md -> PDF..."
      pandoc "$SCRIPT_DIR/$md" \n        -o "$OUT_DIR/${md%.md}.pdf" \n        --pdf-engine=xelatex \n        -V mainfont="$CJK_FONT" \n        -V geometry:margin=2cm \n        -V fontsize=11pt \n        --toc \n        -V toc-title="목차"
    done

    # 교육 슬라이드는 Marp CLI로 변환
    if command -v marp &> /dev/null; then
      echo "Converting training-slides.md -> PDF (Marp)..."
      marp "$SCRIPT_DIR/training-slides.md" \n        --pdf --allow-local-files \n        -o "$OUT_DIR/training-slides.pdf"
    else
      echo "Marp CLI 미설치 — pandoc beamer로 대체 변환"
      pandoc "$SCRIPT_DIR/training-slides.md" \n        -t beamer \n        -o "$OUT_DIR/training-slides.pdf" \n        --pdf-engine=xelatex \n        -V mainfont="$CJK_FONT" \n        -V theme=metropolis
    fi

    echo "PDF 생성 완료: $OUT_DIR/"
    ls -la "$OUT_DIR"/*.pdf
    ```
