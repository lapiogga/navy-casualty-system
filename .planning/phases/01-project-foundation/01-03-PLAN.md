1. **Dockerfile** (RESEARCH.md "멀티스테이지 Dockerfile" 참조):
```dockerfile
# Build stage
FROM eclipse-temurin:21-jdk-alpine AS builder
WORKDIR /app
COPY backend/gradle/ gradle/
COPY backend/gradlew backend/settings.gradle.kts backend/build.gradle.kts ./
RUN ./gradlew dependencies --no-daemon
COPY backend/src/ src/
RUN ./gradlew bootJar --no-daemon

# Frontend build stage
FROM node:22-alpine AS frontend-builder
WORKDIR /app
COPY frontend/package*.json ./
RUN npm ci
COPY frontend/ ./
RUN npm run build

# Runtime stage
FROM eclipse-temurin:21-jre-alpine
WORKDIR /app
RUN addgroup -S appgroup && adduser -S appuser -G appgroup
COPY --from=builder /app/build/libs/*.jar app.jar
COPY --from=frontend-builder /app/dist/ /app/static/
USER appuser
EXPOSE 8080
ENTRYPOINT ["java", "-jar", "app.jar"]
```

2. **docker-compose.yml** (RESEARCH.md "Docker Compose 기본 구성" 참조):
- app 서비스: build context=., dockerfile=Dockerfile, port=${APP_PORT:-8080}:8080
- 환경변수: SPRING_DATASOURCE_URL, SPRING_DATASOURCE_USERNAME, SPRING_DATASOURCE_PASSWORD, SPRING_PROFILES_ACTIVE
- depends_on db with condition: service_healthy
- db 서비스: postgres:16-alpine, POSTGRES_DB/USER/PASSWORD, volume postgres_data
- healthcheck: pg_isready, interval 10s, timeout 5s, retries 5
- 둘 다 restart: unless-stopped

3. **docker-compose.override.yml** (로컬 개발용, per D-07):
```yaml
services:
  app:
    build:
      target: builder
    volumes:
      - ./backend/src:/app/src
    environment:
      SPRING_PROFILES_ACTIVE: dev
    ports:
      - "5005:5005"
    command: ["java", "-agentlib:jdwp=transport=dt_socket,server=y,suspend=n,address=*:5005", "-jar", "app.jar"]

  db:
    ports:
      - "${DB_PORT:-5432}:5432"
```

4. **docker-compose.prod.yml** (에어갭 운영 배포, per D-07 + INFRA-05 스텁):
```yaml
services:
  app:
    restart: always
    environment:
      SPRING_PROFILES_ACTIVE: prod
    volumes:
      - ./certs:/app/certs:ro
    # INFRA-05: TLS 활성화는 Phase 8에서 아래 환경변수 설정
    # SERVER_SSL_KEY_STORE: /app/certs/keystore.p12
    # SERVER_SSL_KEY_STORE_PASSWORD: ${SSL_KEYSTORE_PASSWORD}
    # SERVER_SSL_KEY_STORE_TYPE: PKCS12

  db:
    # 운영 환경에서는 외부 포트 노출하지 않음
    ports: !reset []
```
